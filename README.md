# Subscription tracker

Приложение для отслеживания подписок

## Начало работы

Скопируйте файл .env.example в .env и заполните в нем переменные. Затем сгенерируйте клиенты для Prisma и запустит dev сервер, выполнив команды:

```bash
npm run prisma:generate
npm run dev
```
